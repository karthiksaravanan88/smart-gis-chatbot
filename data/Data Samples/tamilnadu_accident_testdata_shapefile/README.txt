Tamil Nadu Accident Test Dataset (Synthetic)
========================================

This is a synthetic testing dataset containing 500 accident point records in Tamil Nadu.
It is generated for software testing, demos, training, and development purposes only.

Coordinate System: WGS 84 (EPSG:4326)
Geometry Type: Point

Files included:
- tn_accidents_500.shp
- tn_accidents_500.shx
- tn_accidents_500.dbf
- tn_accidents_500.prj
- tn_accidents_500.cpg
- tn_accidents_500.csv

Key fields:
- acc_id: Accident ID
- acc_date: Accident date
- acc_time: Accident time
- district: District name
- severity: Minor / Major / Fatal
- vehicles: Main vehicle category
- roadtype: Road category
- weather: Weather condition
- lightcond: Lighting condition
- fatals: Number of fatalities
- injuries: Number of injuries
- cause: Simulated primary cause
- lon / lat: Coordinates
